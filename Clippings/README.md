# Clippings

`Clippings` holds imported source material that belongs in Sourcebook but still needs cleanup, extraction, or routing.

## What Belongs Here

- one source-specific clipping per file
- web articles, copied references, and source imports worth preserving
- material that may later become `Notes`, `Learning & Skills`, `Bahas Soal`, `Personal Advice`, or `Collected Readings`

## What Does Not

- general capture that belongs in `D:\Imperial Records\Inbox`
- active project research that belongs beside the project in `D:\Imperial Records\Projects`
- polished notes that already belong in another Sourcebook folder
- attachment-only dumps without a markdown note explaining the source

## How To Use It

- preserve enough metadata to trace the original source
- remove obvious clipper noise before promoting a note
- keep source-heavy text here until the durable idea has a clearer home
- move extracted knowledge into the right Sourcebook bucket instead of duplicating it across folders

## Review Rhythm

Review clippings in batches. Clean and route the strongest sources; delete or ignore imports that no longer justify attention.
