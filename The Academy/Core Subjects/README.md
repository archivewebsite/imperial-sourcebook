# Core Subjects

`Core Subjects` stores the main academic skill domains that support study across the rest of `The Academy`.

## Current Buckets

- `Critical Reasoning`
- `Quantitative Foundations`
- `Textual Analysis`

## What Belongs Here

- subject branches that deserve their own internal structure
- shared academic foundations rather than one-off article notes

## How To Use It

- keep each child folder focused on one discipline
- use this folder as the map, not the place for every detailed note
