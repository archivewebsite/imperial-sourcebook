# Spaced Repetition Source

`Spaced Repetition Source` stores notes that are good candidates for drills, flashcards, or repeated retrieval practice.

## What Belongs Here

- facts, formulas, vocab, or definitions worth revisiting
- source notes that can later be converted into cards or prompts

## What Does Not

- broad essays that are not easy to rehearse actively

## How To Use It

- keep the source note atomic enough to become practice material later
- link back to the fuller explanation when needed
