# How I Learned X

`How I Learned X` is for retrospectives on how you learned something well enough to explain the path.

## What Belongs Here

- process notes about what worked
- learning breakdowns that may help you repeat the same progress later
- postmortems on study methods, sequencing, or key insights

## How To Use It

- focus on the path, not just the final knowledge
- capture what was efficient, wasteful, or unexpectedly useful
