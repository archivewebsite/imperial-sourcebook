# Literasi

`Literasi` stores language and literacy notes for entrance-exam preparation.

## What Belongs Here

- notes such as `Afiksasi`, `Kelas Kata`, and `Hubungan Antarkalimat`
- quick-reference material for recurring language question types

## How To Use It

- keep one note per concept, pattern, or recurring exam issue
- make the note titles concrete enough to find quickly during review
