## 1. Pengertian

**Konfigurasi elektron** adalah susunan elektron pada subkulit atom dalam orbital atom atau molekulnya — menggambarkan bagaimana elektron tersebar dan menempati kulit-kulit energi di sekeliling inti atom.

**Struktur atom (ringkas):**
- **Inti atom** → mengandung **proton** (+) dan **neutron** (netral)
- **Kulit elektron** → tempat **elektron** (−) bergerak mengelilingi inti

---

## 2. Kulit Elektron & Bilangan Kuantum Utama (n)

Kulit elektron diberi nama berdasarkan urutan alfabet dari yang terdekat ke inti:

| Kulit | Nama | Nilai n |
|-------|------|---------|
| Pertama | K | 1 |
| Kedua | L | 2 |
| Ketiga | M | 3 |
| Keempat | N | 4 |
| dst. | O, P, Q… | 5, 6, 7… |

**Prinsip penting:**
- Semakin besar nilai **n** → kulit semakin **jauh dari inti** → energi elektron semakin **besar**
- Pengisian elektron dimulai dari kulit **K** (energi paling rendah)
- **Kapasitas maksimum** elektron per kulit ke-n = **2n²**
- **Kapasitas maksimum kulit terluar** = **8 elektron**

| n | Kulit | Kapasitas Maks (2n²) |
|---|-------|----------------------|
| 1 | K | 2 |
| 2 | L | 8 |
| 3 | M | 18 |
| 4 | N | 32 |

## 3. Bilangan Kuantum

Setiap elektron dalam atom dicirikan oleh **4 bilangan kuantum**:

### 3.1 Bilangan Kuantum Utama (n)
- Mendeskripsikan **ukuran dan tingkat energi** orbital
- Nilai: bilangan bulat positif → **n = 1, 2, 3, 4, …**

### 3.2 Bilangan Kuantum Azimuth (l)
- Mendeskripsikan **bentuk** orbital (subkulit)
- Nilai: bilangan bulat **0 hingga (n − 1)**
- Penamaan subkulit:

| Nilai l | Label Subkulit | Bentuk Orbital |
|---------|---------------|----------------|
| 0 | **s** | Bola simetris |
| 1 | **p** | Dumbbell (seperti dua balon) |
| 2 | **d** | Bunga kompleks |
| 3 | **f** | Sangat kompleks |

### 3.3 Bilangan Kuantum Magnetik (m)
- Mendeskripsikan **orientasi** orbital di ruang
- Nilai: bilangan bulat **−l hingga +l**
- Jumlah orbital per subkulit = **2l + 1**

### 3.4 Bilangan Kuantum Spin (s)
- Mendeskripsikan **arah spin** elektron dalam orbital
- Nilai: **+½** (spin ke atas ↑) atau **−½** (spin ke bawah ↓)

### Tabel Kombinasi Bilangan Kuantum (4 Kulit Pertama)

| n | Kulit | l | Subkulit | Nilai m | Jml Orbital | Jml e⁻ maks di kulit |
|---|-------|---|----------|---------|-------------|----------------------|
| 1 | K | 0 | 1s | 0 | 1 | 2 |
| 2 | L | 0 | 2s | 0 | 1 | 8 |
|   |   | 1 | 2p | −1, 0, +1 | 3 | |
| 3 | M | 0 | 3s | 0 | 1 | 18 |
|   |   | 1 | 3p | −1, 0, +1 | 3 | |
|   |   | 2 | 3d | −2, −1, 0, +1, +2 | 5 | |
| 4 | N | 0 | 4s | 0 | 1 | 32 |
|   |   | 1 | 4p | −1, 0, +1 | 3 | |
|   |   | 2 | 4d | −2, −1, 0, +1, +2 | 5 | |
|   |   | 3 | 4f | −3, −2, −1, 0, +1, +2, +3 | 7 | |

## 5. Tingkatan Energi Subkulit (Diagram Aufbau)

Urutan pengisian subkulit berdasarkan **tingkat energi naik** (dari rendah ke tinggi):

```
1s
2s  2p
3s  3p  3d
4s  4p  4d  4f
5s  5p  5d
6s  6p
7s
```

**Cara membaca:** ikuti arah panah diagonal dari kiri atas ke kanan bawah:

```
1s → 2s → 2p → 3s → 3p → 4s → 3d → 4p → 5s → 4d → 5p → 6s → 4f → 5d → 6p → 7s …
```

> **Catatan penting:** Subkulit 4s memiliki energi **lebih rendah** dari 3d, sehingga 4s diisi **lebih dahulu** dari 3d.

## 6. Tiga Aturan Penulisan Konfigurasi Elektron

### 6.1 Asas Aufbau *(Aufbau Principle)*
> Elektron mengisi orbital dari tingkat energi **terendah ke tertinggi**.

**Contoh:**
```
₁₂Mg = 1s² 2s² 2p⁶ 3s²
₂₃V  = 1s² 2s² 2p⁶ 3s² 3p⁶ 4s² 3d³
```

---

### 6.2 Aturan Hund *(Hund's Rule)*
> Elektron mengisi orbital-orbital **dengan tingkat energi sama** secara **satu per satu** terlebih dahulu (spin searah), baru kemudian berpasangan.

**Ilustrasi untuk ₇N = 1s² 2s² 2p³:**

| | 1s² | 2s² | 2p³ |  |  |
|---|---|---|---|---|---|
| (i) — ❌ Salah | ↑↓ | ↑↓ | ↑↓ | ↑ | □ |
| (ii) — ✅ Benar | ↑↓ | ↑↓ | ↑ | ↑ | ↑ |

Susunan (ii) benar karena elektron mengisi **semua orbital dulu** sebelum berpasangan.

**Ilustrasi untuk ₈O = 1s² 2s² 2p⁴:**

| | 1s² | 2s² | 2p⁴ |  |  |
|---|---|---|---|---|---|
| ✅ Benar | ↑↓ | ↑↓ | ↑↓ | ↑ | ↑ |

Pada 2p⁴: dua orbital diisi penuh (↑↓), dua orbital diisi satu-satu (↑ ↑). Tidak ada orbital kosong jika masih ada elektron yang bisa mengisi.

---

### 6.3 Larangan Pauli *(Pauli Exclusion Principle)*
> Tidak boleh ada dua elektron dalam satu atom yang memiliki **keempat bilangan kuantum (n, l, m, s) yang sama**.
> 
> → Jika dua elektron menempati **orbital yang sama**, maka spin keduanya **harus berlawanan** (+½ dan −½).

**Konsekuensi:** Setiap orbital hanya dapat menampung **maksimum 2 elektron** (dengan spin berlawanan).

**Contoh pada orbital 2p⁴** (elektron ke-5 hingga ke-8):

| Elektron ke- | n | l | m  | s   |
|-------------|---|---|----|-----|
| 5 | 2 | 1 | −1 | +½ |
| 6 | 2 | 1 |  0 | +½ |
| 7 | 2 | 1 | +1 | +½ |
| 8 | 2 | 1 | −1 | −½ |

Keempat bilangan kuantum tiap elektron **berbeda satu sama lain** ✓

## 7. Konfigurasi Elektron Gas Mulia

Gas mulia (Golongan VIIIA) memiliki konfigurasi elektron yang **sangat stabil** (kulit terluar penuh). Terdiri dari:

| Unsur | Simbol | Nomor Atom | Konfigurasi Elektron |
|-------|--------|-----------|----------------------|
| Helium | He | 2 | 1s² |
| Neon | Ne | 10 | 1s² 2s² 2p⁶ |
| Argon | Ar | 18 | 1s² 2s² 2p⁶ 3s² 3p⁶ |
| Kripton | Kr | 36 | 1s² 2s² 2p⁶ 3s² 3p⁶ 4s² 3d¹⁰ 4p⁶ |
| Xenon | Xe | 54 | 1s² 2s² 2p⁶ 3s² 3p⁶ 4s² 3d¹⁰ 4p⁶ 5s² 4d¹⁰ 5p⁶ |
| Radon | Rn | 86 | 1s² 2s² 2p⁶ 3s² 3p⁶ 4s² 3d¹⁰ 4p⁶ 5s² 4d¹⁰ 5p⁶ 6s² 4f¹⁴ 5d¹⁰ 6p⁶ |

## 8. Penyingkatan dengan Notasi Gas Mulia

Konfigurasi elektron yang panjang dapat **disingkat** menggunakan simbol gas mulia terdekat yang berada **sebelum** unsur tersebut dalam tabel periodik.

**Cara:** Ganti bagian konfigurasi yang sama dengan konfigurasi gas mulia menggunakan **[simbol gas mulia]**.

**Contoh:**
```
₂₁Sc (lengkap) = 1s² 2s² 2p⁶ 3s² 3p⁶ 4s² 3d¹
₂₁Sc (singkat)  = [Ar] 4s² 3d¹
```
*(karena konfigurasi ₁₈Ar = 1s² 2s² 2p⁶ 3s² 3p⁶)*

**Contoh lain:**
```
₂₆Fe  = [Ar] 4s² 3d⁶
₃₅Br  = [Ar] 4s² 3d¹⁰ 4p⁵
₄₇Ag  = [Kr] 5s¹ 4d¹⁰   ← anomali (lihat bagian 9)
```

## 9. Konfigurasi Elektron Unsur Transisi — Anomali

Beberapa unsur transisi **menyimpang** dari urutan Aufbau biasa karena konfigurasi **setengah penuh (d⁵)** atau **penuh (d¹⁰)** lebih stabil.

| Unsur | Konfigurasi "Seharusnya" | Konfigurasi Aktual | Alasan |
|-------|--------------------------|-------------------|--------|
| ₂₄Cr | [Ar] 4s² 3d⁴ | **[Ar] 4s¹ 3d⁵** | 3d⁵ lebih stabil (setengah penuh) |
| ₂₉Cu | [Ar] 4s² 3d⁹ | **[Ar] 4s¹ 3d¹⁰** | 3d¹⁰ lebih stabil (penuh) |

## 10. Penentuan Golongan & Periode dari Konfigurasi Elektron

| Info dari konfigurasi | Yang ditentukan |
|----------------------|-----------------|
| Nomor kulit terbesar (n terluar) | **Periode** |
| Elektron valensi pada subkulit s dan p (golongan utama A) | **Golongan** |
| Jumlah elektron pada subkulit (n−1)d + ns (golongan transisi B) | **Golongan** |

**Contoh:**
- ₁₁Na = [Ne] 3s¹ → Periode **3**, Golongan **IA**
- ₁₇Cl = [Ne] 3s² 3p⁵ → Periode **3**, Golongan **VIIA**
- ₂₆Fe = [Ar] 4s² 3d⁶ → Periode **4**, Golongan **VIII B** (3d⁶ + 4s² = 8 elektron)

---

## 11. Rangkuman: Tiga Aturan Utama

| Aturan | Isi | Dampak |
|--------|-----|--------|
| **Aufbau** | Isi dari energi rendah ke tinggi | Urutan pengisian subkulit |
| **Hund** | Isi satu-satu dulu, baru berpasangan | Distribusi elektron dalam orbital satu subkulit |
| **Larangan Pauli** | Tidak ada dua elektron dengan 4 bilangan kuantum sama | Maks. 2 elektron per orbital, spin harus berlawanan |

## 12. Contoh Soal & Pembahasan

### Soal 1 — Konfigurasi Elektron Dasar
**Tentukan konfigurasi elektron ₁₅P!**

Jawab:
```
₁₅P = 1s² 2s² 2p⁶ 3s² 3p³
     = [Ne] 3s² 3p³
```
- Periode: 3 (kulit terluar n=3)
- Golongan: VA (5 elektron valensi: 3s² 3p³)

---

### Soal 2 — Bilangan Kuantum
**Tentukan 4 bilangan kuantum elektron terakhir pada ₁₅P!**

Jawab: Elektron terakhir berada di subkulit 3p³ (orbital ketiga dari 3p).
- n = 3
- l = 1 (subkulit p)
- m = +1 (orbital ketiga: −1, 0, **+1**)
- s = +½ (belum berpasangan, spin pertama)

---

### Soal 3 — Aturan Hund
**Gambarkan diagram orbital untuk ₇N = 1s² 2s² 2p³!**

```
1s²    2s²    2p³
[↑↓]  [↑↓]  [↑][↑][↑]
```
Ketiga elektron 2p masing-masing menempati satu orbital dengan spin sejajar → memenuhi Aturan Hund ✓
